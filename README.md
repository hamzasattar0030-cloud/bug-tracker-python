# Python Bug Tracker CLI

A simple command-line bug tracking tool built using Python to simulate real-world QA defect management and issue tracking.

---

## 🚀 Features

- Add new bugs with title and description  
- Assign priority (Low, Medium, High)  
- Track bug status (Open / Fixed)  
- Automatically generate unique bug IDs  
- Store date created for each bug  
- Save and load data using JSON  

---

## 🛠️ Technologies Used

- Python  
- JSON  
- Command Line Interface (CLI)  

---

## ▶️ How to Run

1. Open terminal in the project folder  
2. Run the application:

py bug_tracker.py
